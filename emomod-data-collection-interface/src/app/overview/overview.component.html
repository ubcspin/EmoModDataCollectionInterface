Study Description

This is a test study to sample the functionality of the data collection interface.

The following study will capture and visualize the following data:

  • Heart Rate (arduino)
  • Respiratory rate (arduino)
  • GSR sensor (arduino)
  • Joystick (arduino)
  • Touch fabric sensor (arduino)

This platform is a successor to the original user interface designed for the TouchTales emotion modelling study. This test case is intended to generalize each of the components by focusing on modularity and adaptability.

The data stream collected for this test case is all arduino-based, however, we intend to extend functionality to WearOS-based wearable sensors.

This platform is created with data privacy guidelines in mind; the platform itself runs locally on a single machine, and is intended to run offline as much as possible.
